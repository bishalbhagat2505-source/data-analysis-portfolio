# Shoe Sales Data Analysis

This project is an Exploratory Data Analysis (EDA) on a Shoe Sales dataset using Python.

In this analysis, I explored the dataset, cleaned the data, and used different charts and graphs to understand sales patterns, customer trends, and brand performance.

---

# Tools and Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

# Steps Performed

## 1. Data Loading
- Loaded the dataset using pandas
- Checked rows, columns, and data types

## 2. Data Cleaning
- Checked missing values
- Removed duplicates
- Fixed column formats if needed

## 3. Univariate Analysis
Performed analysis on single columns using:
- Histograms
- Countplots
- Boxplots

## 4. Bivariate Analysis
Compared different columns using:
- Scatter plots
- Bar charts
- Correlation heatmap

## 5. Business Insights
Found useful insights related to:
- Sales trends
- Brand performance
- Product categories
- Customer purchasing behavior

---

# Some Important Insights

- Some brands had much higher sales compared to others
- Certain shoe types were more popular
- Online and offline sales channels showed different performance
- A few variables had strong positive correlation with sales
